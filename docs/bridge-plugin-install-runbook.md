# Bridge Plugin Install Runbook

## Download und Upload

1. das Paket `dist/staging-only/site-optimizer-bridge-0.1.0-real-staging1-wp-electri-c-ity-studios-24-7-com.zip` lokal vom Server holen
2. im WordPress-Admin der staging-domain `Plugins -> Installieren -> Plugin hochladen`
3. nur das staging-only Paket verwenden

## Nach dem Upload

1. Plugin `Site Optimizer Bridge` in der Plugin-Liste pruefen
2. Version `0.1.0-real-staging1` pruefen
3. Aktivierung nur nach Backup-/Restore-Bestaetigung
