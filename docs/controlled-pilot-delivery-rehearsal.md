# Controlled Pilot Delivery Rehearsal

## Ziel

Die geschuetzte Bereitstellung eines staging-only Pilotpakets wird simuliert, ohne oeffentliche Ausgabe.

## Rehearsal

- package metadata preview pruefen
- entitlement preview pruefen
- manifest preview pruefen
- release artifact preview pruefen
- Delivery nur als protected-preview denken

## Gate

- Controlled Pilot Delivery Rehearsal: `approval_required`
