# Customer Install Pack Model

## Zweck

Dieses Dokument beschreibt das lokale Install-Pack-Artefakt fuer eine spaetere geschuetzte Kauf-/Fulfillment-Schicht.

## Inhalt

- Paketreferenz und Checksumme
- Lizenzobjektpfad
- Manifestpfad
- Entitlementpfad
- Rollback-Profilpfad
- Validation-Checklist-Pfad
- Install-Runbook
- Safe-Mode-Test-Runbook
- kundenlesbare Sicht auf Domain, Scope, Dokumentation und geschuetzten Downloadstatus
- spaetere Referenzen auf License-Issuance-Prep und Signed-Delivery-Prep

## Nicht enthalten

- offene Delivery
- offener Login
- offene API-Freigabe
- automatischer Live-Rollout
