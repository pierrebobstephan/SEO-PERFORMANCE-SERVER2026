# First Plugin Pilot Boundaries

## Ziel

Der erste echte Pilot bleibt eng, defensiv und staging-orientiert.

## Grenzen

- `observe_only` oder `safe_mode` als Default
- kein siteweiter Eingriff
- keine automatische Rank-Math-Ablosung
- keine offene Multi-Domain-Wirkung
- nur Homepage Meta / Head / Structure / visibility diagnostics

## Nicht erlaubt

- unklarer Scope
- fehlender Rollback-Pfad
- fehlendes Coexistence-Bild
- Produktivtest auf ungeschuetzter Live-Site

## Gate

- First Plugin Pilot: `approval_required`
