# First Real Product Test Checklist

## Vorbedingungen

- Testdomain bestaetigt
- Lizenzbindung bestaetigt
- Scope bestaetigt
- Backup abgeschlossen
- Rollback vorbereitet
- Coexistence-Bild dokumentiert

## Durchfuehrung

- Plugin installiert
- `safe_mode` aktiv
- erste Diagnostics gesammelt
- Protected Delivery nicht nach aussen geoeffnet

## Nachbereitung

- Ergebnisse sichern
- Learnings dokumentieren
- Entscheidung: weiter beobachten, anpassen oder abbrechen

## Gate

- First Real Product Test Checklist: `approval_required`
