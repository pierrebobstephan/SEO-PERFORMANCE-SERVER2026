# First Real Product Test Stop Conditions

## Sofortiger Abbruch

- Backup fehlt oder ist unpruefbar
- Restore-Pfad unklar
- Domain- oder Scope-Mismatch
- Konfliktbild unklar
- unerwartete aktive Ausgabe
- protected route exposure

## Rollback-Ausloeser

- Primarmetriken kippen
- neue Fehler erscheinen
- Scope wird verletzt
- Kausalitaet bleibt unklar

## Gate

- First Real Product Test Stop Conditions: `approval_required`
