# First Real Product Test Success Metrics

## Primarmetriken

- Plugin bleibt stabil installierbar
- `safe_mode` / `observe_only` funktioniert wie vorgesehen
- keine Doppel-Ausgabe
- kein unerwarteter Scope-Austritt

## Sekundaermetriken

- Diagnostics sind plausibel
- Theme/Builder/SEO-Plugin-Konflikte werden sauber erkannt
- Rollback kann reproduzierbar beschrieben werden

## Nicht als Erfolg zaehlt

- irgendeine aggressive Live-Wirkung ohne Validierung
- Scope-Erweiterung ohne Freigabe

## Gate

- First Real Product Test Success Metrics: `approval_required`
