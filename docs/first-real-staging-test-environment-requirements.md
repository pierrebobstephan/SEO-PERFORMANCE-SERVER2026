# First Real Staging Test Environment Requirements

## Benoetigte Eingangsdaten

- WordPress version
- active theme
- active builder
- active SEO plugin
- plugin inventory
- backup confirmation
- restore confirmation
- rollback owner
- validation owner

## Technische Mindestbedingungen

- Install/Deactivate/Delete-Rechte auf der staging-/testfaehigen WordPress-Umgebung
- dokumentierte Backup-Basis vor Test
- dokumentierter Restore-Pfad vor Test
- keine ungesicherte Produktivseite

## Gate

- First Real Staging Test Environment Requirements: `approval_required`
