# First Real Staging Test Scope V1

## Erlaubter Startscope

- homepage meta diagnostics
- head diagnostics
- structure diagnostics
- visibility diagnostics

## Nicht Teil von V1

- siteweite Ausweitung
- automatische Title-, Canonical- oder Robots-Uebernahme
- automatische Rank-Math-Ablosung
- Theme- oder Builder-Manipulation
- globale Performance- oder Cache-Eingriffe

## Scope-Regel

- wenn Ownership unklar ist, bleibt der Test in `observe_only`
- wenn Konflikte unklar sind, bleibt der Test in `safe_mode`

## Gate

- First Real Staging Test Scope V1: `approval_required`
