# First Test Stop Conditions

## Sofortiger Abbruch

- Fatal Error
- sichtbare Seitenschaedigung
- Konflikt mit aktivem Theme, Builder oder SEO-Plugin
- Domain- oder Scope-Mismatch
- Rollback-Pfad nicht mehr intakt

## Kein Fortschritt

- Ownership bleibt unklar
- Diagnostics sind nicht plausibel
- Validierung bleibt unvollstaendig

## Gate

- First Test Stop Conditions: `approval_required`
