# Local Renewal Prep Model

## Zweck

Dieses Dokument beschreibt die lokale Vorbereitung fuer spaetere Verlaengerung einer gebundenen Lizenz.

## Regeln

- bleibt `approval_required`
- nutzt weiter `PayPal Business` nur als modellierte Zahlungsmethode
- verlaengert keine Entitlements automatisch
- oeffnet keine Delivery oder Login-Funktion
