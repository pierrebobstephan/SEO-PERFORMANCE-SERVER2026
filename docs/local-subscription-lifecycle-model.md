# Local Subscription Lifecycle Model

## Zweck

Dieses Dokument beschreibt die lokale, geschuetzte Vorbereitung fuer Renewal, Grace und Failed-Payment-Recovery im spaeteren Abo-Lebenszyklus.

## Bestandteile

- Subscription Lifecycle Prep
- Renewal Prep
- Failed Payment Recovery Prep

## Grenzen

- keine echte Billing-Engine
- keine echte Renewal-Ausgabe
- keine automatische Kundenfreigabe
- keine sichtbare Site-Manipulation bei Zahlungsausfall
