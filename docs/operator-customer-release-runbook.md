# Operator Customer Release Runbook

## Ziel

Dieses Runbook beschreibt die lokale, geschuetzte Operator-Pruefung vor einer spaeteren Kundenfreigabe.

## Schritte

1. Payment-Confirmation und Rechnungsstatus pruefen.
2. License-Issuance-Prep und Signed-Delivery-Prep pruefen.
3. Install-Pack, Support-Handover und Rollback-Ownership pruefen.
4. Customer-Release-Authorization nur nach positiver Go/No-Go-Entscheidung weiterfuehren.

## Grenzen

- keine oeffentliche Bereitstellung
- keine Customer-Self-Service-Freigabe
- keine Scope-Ausweitung ueber die gebundene Domain
