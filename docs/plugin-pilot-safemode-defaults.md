# Plugin Pilot Safe Mode Defaults

## Default-Haltung

- Start in `safe_mode` oder `observe_only`
- bei Konflikten deaktiviert bleiben
- bei unklarer Ownership deaktiviert bleiben
- bei fehlender Lizenz-/Domain-Klarheit nicht aktiv werden

## Erst spaeter moegliche Eskalation

- nach Backup
- nach Scope-Bestaetigung
- nach Rollback-Freigabe
- nach Validierungsdefinition

## Gate

- Plugin Pilot Safe Mode Defaults: `approval_required`
