# Plugin Pilot Validation and Rollback

## Vor Pilotanwendung

- Before-State sichern
- Konfliktbild sichern
- Rollback-Schritte vorbereiten
- Abort-Kriterien definieren

## Nach Pilotstart

- Sichtpruefung
- Head-/Meta-/Structure-Checks
- Fehler- und Konfliktsignale
- unmittelbare Rueckfallmoeglichkeit

## Rollback-Pflicht

- bei Regression
- bei unklarer Kausalitaet
- bei Doppel-Ausgabe
- bei Scope-Verletzung

## Gate

- Plugin Pilot Validation and Rollback: `approval_required`
