# Post Purchase Customer Visibility Layer

## Ziel

- nach Kauf und Installation lokale, gut lesbare Suite-Daten im Plugin bereitstellen
- keine neue Customer-API oder oeffentliche Route erzeugen

## Sichtbare Bloecke

- License / Domain / Scope Panel
- Installation Health Signals
- Production Cutover Checklist

## Grenzen

- nur Admin-intern
- nur domain-bound pro installierter Instanz
- weiter staging-only oder spaeter lizenzierter lokaler Pfad
