# Post-Test Decision Gates

## Entscheidungen

- `passed` -> weiterer staging Ausbau moeglich
- `inconclusive` -> in staging bleiben, Daten vertiefen
- `failed` -> rollback and redesign

## Keine Freigabe

- kein Uebergang in Referenzsystem-Pilot ohne bestandenes Rollback-Rehearsal
- kein Uebergang ohne ausreichend klares Coexistence-Bild

## Gate

- Post-Test Decision Gates: `approval_required`
