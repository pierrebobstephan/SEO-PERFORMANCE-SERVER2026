# Pre-Test Backup and Rollback Checklist

## Pflicht vor Realtest

- WordPress-Dateien gesichert
- Datenbank gesichert
- Plugin- und Theme-Inventar gesichert
- Rollback-Verantwortung benannt
- Restore-Prozedur dokumentiert

## Pflicht fuer Testfreigabe

- kein Test ohne Backup
- kein Test ohne Restore
- kein Test ohne klaren Scope
- kein Test ohne Abort-Kriterien

## Gate

- Pre-Test Backup and Rollback Checklist: `approval_required`
