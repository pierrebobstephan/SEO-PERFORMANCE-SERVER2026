# Protected Pilot Path Model

## Ziel

Ein spaeter real nutzbarer Pilotpfad wird vorbereitet, bleibt aber strikt nicht-oeffentlich.

## Bestandteile

- package preview
- license/domain match preview
- pilot manifest preview
- installability preview

## Schutzgrenzen

- keine oeffentliche Route
- keine offene Download-Ausgabe
- keine Login- oder Customer-Flaeche
- nur lokal oder spaeter hinter geschuetzter Operator-/Preview-Schicht

## Gate

- Protected Pilot Path: `approval_required`
