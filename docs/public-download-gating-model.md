# Public Download Gating Model

## Zweck

Dieses Dokument beschreibt den gated Download-Zugang ueber das oeffentliche Portal.

## Grundsatz

- keine anonyme offene Plugin-Auslieferung mit produktiver Wirkung
- Downloadzugang spaeter nur ueber Lizenz, Domain-Entitlement, Kanal und Approval

## Oeffentlich erlaubt

- Download-Seite als Informationsseite
- Hinweis auf gated Zugang
- Beschreibung des Lizenz- und Entitlement-Modells

## Nicht oeffentlich

- offene Download-URLs
- offene Lizenz-API
- ungeschuetzte Kunden- oder Operator-Endpunkte

## Status

- Download-Gating-Modell: `blueprint_ready`
- echte Download-Freigabe: `approval_required`
