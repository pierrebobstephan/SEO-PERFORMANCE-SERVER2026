# Public Portal Buy Page Comparison Logic

## Zweck

Die Vergleichslogik auf `/buy` soll die drei oeffentlichen Tiers sichtbar machen, ohne Mehrdeutigkeiten bei Domain-Scope oder Delivery zu erzeugen.

## Sichtbare Planlogik

- `Guardian Core Monthly`
  - `EUR 2,500.00 / month`
  - `Monthly optimization scope for one exact licensed domain.`
  - `Includes documentation and licensed download access.`
- `Sovereign Enterprise Monthly`
  - `EUR 9,500.00 / month`
  - `Monthly multi-domain optimization for an exact licensed domain set confirmed during commercial review.`
  - `Includes documentation and licensed download access.`
- `Sovereign Critical Monthly`
  - `EUR 25,000.00 / month`
  - `Monthly high-compliance scope for an individually approved exact licensed domain set.`
  - `Includes documentation and licensed download access.`

## Vergleichsregeln

- Preis, exact licensed scope und Included-Satz bleiben pro Tier sofort sichtbar
- Short public copy bleibt direkt am Tier
- keine zusaetzlichen Leistungszusagen
- keine unscharfe Multi-Domain-Sprache
- Delivery bleibt als getrennte, geschuetzte Stufe erklaert
