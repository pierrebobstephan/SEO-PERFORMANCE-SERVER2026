# Public Portal Canonical and Breadcrumb Plan

## Canonicals

- jede oeffentliche Seite referenziert sich selbst
- `/impressum` kanonisiert auf `/legal`
- geschuetzte Routen erhalten keine oeffentlichen Canonical-Ziele

## Breadcrumbs

- alle Nicht-Root-Seiten zeigen Breadcrumbs
- Breadcrumb-Labels folgen der oeffentlichen IA
- Agencies und Publishers erhalten eigene Breadcrumb-Titel
