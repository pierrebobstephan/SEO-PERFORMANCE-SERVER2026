# Public Portal Crosslink Consistency Plan

## Ziel

Die oeffentliche IA soll ohne Widerspruch zwischen Produkt, Pricing, Licensing, Downloads, Support und Legal funktionieren.

## Pflichtpfade

- Home -> Buy / Explore / Licensing / Downloads / Support / Terms
- Explore -> Buy / Features / Docs / Licensing / Downloads / Support
- Buy -> Terms / Licensing / Downloads / Support / Refund
- Licensing -> Buy / Terms / Downloads / Support
- Downloads -> Buy / Licensing / Support / Terms
- Support -> Buy / Downloads / Licensing / Contact

## Grenzen

- keine Links auf geschuetzte Routen
- keine CTA auf Login, Checkout oder offene Delivery
