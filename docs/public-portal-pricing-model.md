# Public Portal Pricing Model

## Zweck

Dieses Dokument definiert das sichtbare Monatspreismodell des Public Portals.

## Oeffentliche Tiers

- `Guardian Core Monthly`: EUR 2,500.00 / month
- `Sovereign Enterprise Monthly`: EUR 9,500.00 / month
- `Sovereign Critical Monthly`: EUR 25,000.00 / month

## Pflichtelemente pro Tier

- exact licensed scope
- documentation
- licensed download access

## Grenzen

- kein Checkout
- keine offene Download-Ausgabe
- keine Fulfillment- oder SLA-Erweiterung
- keine Reseller-, White-Label- oder offenen Multi-Domain-Rechte
