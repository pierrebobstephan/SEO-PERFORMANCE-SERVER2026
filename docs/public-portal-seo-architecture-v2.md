# Public Portal SEO Architecture V2

## Zweck

Dieses Dokument beschreibt die portalweite SEO-Architektur nach der Audience- und Buy-Erweiterung.

## Leitlinien

- jede oeffentliche Seite hat ein klares semantisches Thema
- Canonicals und Breadcrumbs bleiben stabil
- strukturierte Daten nur dort, wo sie inhaltlich passen
- interne Verlinkung stuetzt Orientierung statt Keyword-Stuffing
- Legal-Seiten bleiben source-synced und werden nicht unnnoetig narrativ veraendert
