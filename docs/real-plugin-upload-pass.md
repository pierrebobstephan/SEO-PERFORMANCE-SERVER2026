# Real Plugin Upload Pass

## Ziel

Den echten Upload im WordPress-Admin staging-only reproduzierbar machen.

## Pass

1. ZIP lokal bereitstellen
2. in `Plugins -> Installieren -> Plugin hochladen` auswaehlen
3. Upload ohne Fatal Error abschliessen
4. Plugin-Dateien und Version gegenpruefen

## Gate

- Real Plugin Upload Pass: `approval_required`
