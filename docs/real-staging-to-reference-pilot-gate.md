# Real Staging To Reference Pilot Gate

## Uebergang erst wenn

- URL-Normalisierung bestanden
- stabiler Safe-Mode-Test bestanden
- Rollback-Rehearsal bestanden
- Coexistence-Lage klar genug
- Scope weiter eng und kontrolliert

## Kein Uebergang wenn

- `localhost`-Referenzen bleiben
- Ownership unklar bleibt
- Staging-Test nur inconclusive oder failed ist
