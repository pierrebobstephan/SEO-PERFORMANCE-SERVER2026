# Reference System Pilot Entry Criteria

## Eintritt nur wenn

- staging-only Testlauf erfolgreich
- keine Fatal Errors
- keine sichtbare Seitenschaedigung
- kein ungeklaerter Konflikt
- Rollback reproduzierbar
- Domain-/Scope-Bindung klar

## Weiterhin Pflicht

- enger Pilot-Scope
- dokumentierter Before-/After-Vergleich
- enger Blast Radius

## Gate

- Reference System Pilot Entry Criteria: `approval_required`
