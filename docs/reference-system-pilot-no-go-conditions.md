# Reference System Pilot No-Go Conditions

## No-Go

- staging-Test nicht bestanden
- Rollback-Rehearsal nicht bestanden
- Coexistence-Fall unklar
- Scope zu breit
- Backup-/Restore-Lage unklar
- Ownership der Head-/Meta-Ausgabe unklar

## Folge

- in staging bleiben
- rollback and redesign
- keine Referenzsystem-Freigabe
