# Single Source Head Meta Checklist

## Muss fuer gruen gelten

- nur eine belastbare Homepage Meta Description Ownership
- keine doppelte Head-/Meta-Ausgabe
- keine unklare SEO-Plugin-Koexistenz
- Operator-Bestaetigung dokumentiert

## Darf nicht passieren

- automatische Ownership-Uebernahme
- Rank-Math-Ablosung
- siteweite Head-Rewrites
