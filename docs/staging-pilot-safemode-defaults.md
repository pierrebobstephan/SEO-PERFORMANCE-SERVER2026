# Staging Pilot Safe Mode Defaults

## Default

- Start in `observe_only` oder `safe_mode`
- bei Konfliktverdacht deaktiviert bleiben
- bei unklarer Ownership deaktiviert bleiben
- bei Domain- oder Lizenz-Mismatch nicht aktiv werden

## Erst spaeter moegliche Eskalation

- nur nach bestandenem Staging-Test
- nur nach Rollback-Rehearsal
- nur nach bestaetigter Coexistence-Lage

## Gate

- Staging Pilot Safe Mode Defaults: `approval_required`
