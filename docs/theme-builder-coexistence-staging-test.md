# Theme Builder Coexistence Staging Test

## Ziel

Theme- und Builder-Koexistenz soll in staging geprueft werden, bevor irgendeine Scope-Ausweitung denkbar wird.

## Beobachtungen

- Head-Ausgabe aus Theme oder Builder
- Struktur- und Visibility-Signale
- Plugin-Hooks und Konfliktindikatoren

## Abbruch

- unklare Source Ownership
- sichtbare Layout- oder Head-Regressionssignale

## Gate

- Theme Builder Coexistence Staging Test: `approval_required`
