# WordPress Pilot Environment Checklist

## Vorbedingungen

- bekannte WordPress-Version
- Theme-Inventar
- Builder-Inventar
- aktives SEO-Plugin-Bild
- Install/Deactivate/Delete-Zugriff
- Backup vor Test
- Restore-Pfad

## Startzustand

- Plugin startet in `safe_mode` oder `observe_only`
- keine automatische Ownership-Uebernahme
- keine automatische Rank-Math-Ablosung

## Gate

- Environment Checklist: `approval_required`
